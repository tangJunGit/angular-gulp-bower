# angular-gulp-bower

```bash
# 克隆
git clone git@github.com:tangJunGit/angular-gulp-bower.git

# 进入angular-gulp-bower目录
cd angular-gulp-bower

# 添加项目所需全局依赖的库
npm install gulp bower -g

# 添加项目依赖的库
npm install
bower install

# 启动服务
gulp server
```

是否想了解`gulp`的用法，[想了解](https://github.com/tangJunGit/gulp)
